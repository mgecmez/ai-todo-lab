## Hazırlık

### Ortam

- Mobile: `npx expo start` ile çalışıyor (iOS Simülatör önerilir)
- Backend: `dotnet run --urls "http://localhost:5100"` ile ayağa kaldırılmış
- Test öncesinde AsyncStorage tamamen temizlenmiş olmalı (bkz. yöntemler aşağıda)

### AsyncStorage Temizleme

**Yöntem A — Simülatörden sil/yeniden yükle (önerilen):**
Simülatörde uygulamayı uzun bas → "Delete App" → Expo'dan yeniden yükle.

**Yöntem B — Geçici kod:**
`App.tsx` başlangıcına geçici olarak `AsyncStorage.clear()` ekle, bir kez çalıştır, geri al.

### Test Notasyonu

Her bölümde dil durumu şu formatta belirtilir:

```
[Dil: TR / EN]   [AsyncStorage: BOŞ / DOLU]
```
